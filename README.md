# Quantum Native Dojo! 書籍版サポートページ
このGithubリポジトリは、「Quantum Native Dojo! 実装しながら学ぶ量子情報科学と量子コンピュータ」（講談社、2026年4月） https://www.kodansha.co.jp/book/products/0000424240 のサポートページです。

このリポジトリでは、書籍の内容を補足するために、以下の情報や資料を提供します：
- [書籍内で使用したコード](./codes)
  - 書籍内で使用したコードについて、Google Colab上で実行した結果とともに公開します。
  - 一部のコードについては、エラーを修正しています。
- 誤植・訂正表（判明次第、作成して随時更新します）
- 補足ノート（本文のより詳しい説明が必要な場合に、作成して随時更新します）

リポジトリ内のファイルのダウンロード法は、例えばこちらをご参照ください：https://docs.github.com/ja/get-started/start-your-journey/downloading-files-from-github
